# Ascenus Consultancy Services

Modern, responsive website for Ascenus Consultancy Services - a job consultancy/recruitment company.

## Tech Stack

- **Vite** - Fast build tool
- **React 19** - UI framework
- **Tailwind CSS 4** - Styling
- **React Router DOM 7** - Routing

## Features

- 10 fully routed pages
- Responsive mobile-first design
- Sticky navbar with blur effect
- Mobile hamburger menu
- Job filtering (category/experience)
- Premium modern UI/UX

## Pages

| Page | Route | Description |
|------|-------|-------------|
| Home | `/` | Hero, services, featured jobs |
| About Us | `/about` | Company info, stats, values |
| Services | `/services` | 5 service cards |
| Jobs | `/jobs` | 6 jobs with filters |
| Gallery | `/gallery` | Image grid |
| Our Success | `/success` | Stats, testimonials |
| For Employers | `/employers` | Login placeholder |
| Contact | `/contact` | Form + contact info |
| Privacy Policy | `/privacy` | Privacy policy |
| Terms | `/terms` | Terms & conditions |

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
